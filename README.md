Personal Financial Control System

# About (ENG)
- This project is a personal financial control system for managing and tracking transactions. It was developed to practice building a complete application, integrating backend, database, and data manipulation.

Features:
Register income and expenses
Automatic balance calculation
List and view transactions
Delete transactions
Export data to Excel ||
Technologies:
Django (Python)
SQLite
Pandas
PyInstaller
Notes

This project helped reinforce concepts such as backend development, database integration, and data processing.

The application can also be executed through a generated .exe file using PyInstaller, without requiring a Python environment.

# Sobre (PT-BR)
- Este projeto é um sistema de controle financeiro pessoal para registro e gerenciamento de transações. Foi desenvolvido com o objetivo de praticar a construção de uma aplicação completa, integrando backend, banco de dados e manipulação de dados.

Funcionalidades:
Cadastro de receitas e despesas
Cálculo automático do saldo
Listagem e visualização de transações
Exclusão de registros
Exportação para Excel ||
Tecnologias:
Django (Python)
SQLite
Pandas
PyInstaller
Observações

O projeto ajudou a reforçar conceitos de desenvolvimento backend, integração com banco de dados e processamento de dados.

Também é possível executar a aplicação através de um arquivo .exe gerado com PyInstaller, sem necessidade de instalar o Python.
